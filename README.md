# vue-css-waves
VueJS CSS waves

## Install
```shell
npm install --save vue-css-waves
```

## Usage
Just import the `Waves` component where you want to see it !

```html
<template>
    <Waves />
</template>

<script>
	import Waves from 'vue-css-waves'
    import 'vue-css-waves/dist/vue-css-waves.css'

	export default {
		components: {
			Waves
		}
	}
</script>
```
